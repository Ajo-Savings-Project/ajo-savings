# Ajó Savings

Experience the convenience of secure group savings and personalized savings plans with Ajó Savings.
Take control of your finances and unlock a brighter financial future.

## Project Installation

**Setup Client:**
```bash
cd client
yarn
yarn dev
```

**Setup Server:**

```bash
cd server
yarn
```
